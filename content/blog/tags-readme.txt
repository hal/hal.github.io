When adding tags to blog posts, please choose from one of the following tags:

- deployment: Related to deployments top level category
- domain: Related to domain mode only (please do not add a tag for standalone mode!)
- configuration: Related to the configuration top level category
- patching: Related to the runtime top level category
- rbac: Related to the access control top level category
- runtime: Related to the runtime top level category
- security: Related to security settings, elytron and rbac
- settings: Related to the settings

If none of the tags above fit to your blog post, feel free to introduce a new tag and add it to the list above.
